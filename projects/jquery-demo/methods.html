.text()

Get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.

Click the button to retrieve the text in the message above.

.html()

Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.

Click the button to retrieve the html in the message above.

.css()

Get the value of a computed style property for the first element in the set of matched elements or set one or more CSS properties for every matched element.

Click the button to temporarily add a class to the jumbotron sentence.

.attr()

Get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.

Click the button to change the src attribute of the img above.

.val()

Get the current value of the first element in the set of matched elements or set the value of every matched element.

Enter some text into the input below and click the button to retrieve the value of the input.

.addClass()

Adds the specified class(es) to each element in the set of matched elements.

Click the button to add a class of "faded-text" to the message above.

.removeClass()

Remove a single class, multiple classes, or all classes from each element in the set of matched elements.

Click the button to remove the class of "faded-text" from the message above.

.toggleClass()

Add or remove one or more classes from each element in the set of matched elements, depending on either the class’s presence or the value of the state argument.

Click the button to toggle the class of "faded-text" on the message above.

.keypress()

Bind an event handler to the “keypress” JavaScript event, or trigger that event on an element.

Click the 'a' key to animate the message above.
